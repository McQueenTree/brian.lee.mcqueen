static int COLUMN_PAY_FLAG = 0
 COLUMN_X variables are used to designate the column order of the output.
These need to be in the same order as the columns are defined in the gridView.
This allows easier access to finding a column in the code behind. For Example:
as opposed to entering this command: 

e.Row.Cells[0].Attributes.Add("class", "hidden");


this command may be entered: 

e.Row.Cells[VariablesCashManager.COLUMN_PAY_FLAG].Attributes.Add("class", "hidden");


If the columns are reordered, the only code that need to change is the int that is assigned to the variable.
